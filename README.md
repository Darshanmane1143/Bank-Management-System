# Bank Management System

A simple Java bank account application with the following files:

- `BankAccount.java` - contains the bank account class and operations such as deposit, withdraw, and balance inquiry.
- `BankApp.java` - contains the application entry point and sample usage.

## How to run

1. Compile both Java files:
   ```sh
   javac BankAccount.java BankApp.java
   ```

2. Run the application:
   ```sh
   java BankApp
   ```

## Notes

- Generated `.class` files are excluded from version control using `.gitignore`.
- This repository is configured to use the `main` branch.
